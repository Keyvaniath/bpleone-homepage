What you see per team

🏠

Today

Record, last game outcome, next game, games played, today's game card, around-the-league scoreboard, latest news.

📅

Schedule

Full season schedule with upcoming/past tabs, W/L outcomes, broadcast networks, weather forecast for outdoor home games.

📊

Standings

Division and conference standings with your team highlighted. Win %, streak, home/road splits.

📈

Stats

Team season stats grouped by category (general / offense / defense) with league ranks.

🩹

Injuries

Current injury report, severity-coded (Out → Doubtful → Questionable → Probable) with return-date estimates.

👥

Roster + Gamelog

Full roster with position filter. Click any player to see their last 20-game log with stat breakdown.

📰

News

Team-specific articles with thumbnails + league-wide feed. Updated every 5 minutes.

🎮

Fantasy

Sleeper integration — enter your username, browse all your NFL/NBA leagues, see standings and matchups.

🏟️

All Teams Overview

Cross-team glance — last game, next game, live status for every team in one row. Auto-refreshes when any game is live.

Data sources

📡

ESPN unofficial JSON

site.api.espn.com, cdn.espn.com, site.web.api.espn.com — scores, schedules, standings, news, injuries, rosters, team stats, player gamelogs, box scores. No auth.

🎮

Sleeper API

api.sleeper.app — fantasy league lookup by username. No OAuth required for public league data.

☁️

Open-Meteo

Free weather forecasts for outdoor home games — Dodger Stadium, Coliseum, Dedeaux Field. Temperature, wind, precipitation chance per game.

Run it yourself

Clone the repo, pip install -r requirements.txt, streamlit run streamlit_app.py. No API keys needed.
