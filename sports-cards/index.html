Data sources

🔄

eBay sold-listings

Primary price truth — weekly refresh via the same scraper that powers the Pokémon TCG desk. Outlier rejection + median + 30d rolling band. Optional eBay Browse API path bypasses Akamai entirely (set EBAY_OAUTH_TOKEN).

📊

PSA pop reports

Seeded manually per card (Cloudflare blocks scraping). Quarterly refresh. Drives the scarcity sub-score directly.

Zero paid subscriptions. Everything runs on free tiers.

Operational

⏰

Weekly refresh GitHub Action

Sundays 22:00 UTC — pulls latest eBay sold comps, appends a price-history snapshot, commits the refreshed JSON. Sets up trend lines automatically over time.

🔔

Discord alerts

Webhook posts when any card flips into STRONG BUY or out into TRIM/SELL. Embeds include composite + sub-scores + direct eBay link.

📧

Trade-of-the-day email

SMTP digest each morning with the top BUY + runner-up + TRIM warnings, in branded HTML matching the homepage.

💰

Tax helper

Schedule D / Form 8949 generator. Collectibles cap (28% long-term rate, §408(m)(2)(A)) baked in. Separates ST/LT lots automatically.
