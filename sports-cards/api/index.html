Refresh cadence

All three endpoints regenerate inside the same GitHub Action:

Sunday 22:00 UTC weekly →
  live_prices.py (eBay sold-comp scrape)
  → build_signals.py
  → build_stats.py
  → build_rss.py
  → commit docs/* + push

Manual trigger: workflow_dispatch in the repo's Actions tab.
