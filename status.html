Cron schedule

GitHub Actions cron: 0 */4 * * * — every 4 hours at :00 UTC. Total runtime budget: 26 minutes (graceful exit before GH Actions 30min timeout).

The 20-step pipeline

  1. live_prices.py — scrape PriceCharting + PokemonPriceTracker + TCGPlayer + eBay (~25 min)
  2. reconcile_prices.py — multi-rule price reconciler w/ grade-ladder sanity check
  3. auto_data_quality_check.py — DQ A-F grading across 5 issue categories
  4. build_price_diff.py — today's price moves
  5. build_morning_brief.py — daily curated picks
  6. build_price_alerts.py — BUY/SELL triggers at price targets
  7. track_source_reliability.py — per-source accuracy stats
  8. build_daily_recap_v2.py — email-ready daily recap
  9. learn_signal_outcomes.py — log every signal emission
  10. optimize_fw_weights.py — Pearson-correlation retune of FW weights
  11. track_psa_pop_changes.py — daily PSA pop snapshots + signal deltas
  12. build_learning_summary.py — surface weights for homepage
  13. arbitrage_scanner_v2.py — multi-path EV arb scanner
  14. build_trading_journal.py — workbook P/L → JSON
  15. build_past_picks_scoreboard.py — model batting average
  16. build_set_heat.py — per-set composite heat scores
  17. build_sealed_market.py — sealed product scanner
  18. build_email_digest.py — daily HTML email digest
  19. build_weekly_recap.py — weekly recap HTML
  20. build_auto_tweet.py — pre-formatted Twitter + LinkedIn posts
  21. build_set_rotation_predictor.py — momentum-of-momentum per set
  22. build_risk_adjusted_tracker.py — Sharpe-like risk metric
  23. pipeline_self_audit.py — RED/YELLOW/GREEN per output file
  24. build_homepage_feed.py + build_card_data.py + commit + push

Open source

Every script and every output is public. The Pokemon TCG repo is at Keyvaniath/pokemon-tcg-trading-desk. Cron logs are visible in the Actions tab. Audit anything.
